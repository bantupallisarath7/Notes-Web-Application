# Notes App

[GitHub](https://github.com/bantupallisarath7/Notes-Web-Application)

---

## Overview
Notes App is a MERN stack web application that allows users to securely create, manage, and search personal notes with authentication and full CRUD functionality.

---

## Features

- **User**
  - Sign up / log in securely  
  - Create, update, and delete notes  
  - Search and filter notes  
  - Pin important notes  

- **Authentication**
  - JWT-based secure login system  

---

## Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT  

---

## Key Highlights

- Full CRUD operations for notes  
- Real-time search and filtering  
- Clean and responsive UI  
- Secure authentication and data handling  

---

## Installation

```bash
# Clone repo
git clone https://github.com/bantupallisarath7/Notes-Web-Application.git

# Backend
cd backend
npm install
npm run dev

# Frontend
cd ../frontend
npm install
npm run dev
```
